# Yifan Tian — Personal Website

Source for [yifantian07.github.io](https://yifantian07.github.io), the personal academic and project portfolio of Yifan Tian.

The site supports a complete English / Simplified Chinese switch, remembers the visitor's language preference, and adapts the typography for both languages.

## Sections

- Personal profile and education
- Research interests and experience
- Publications
- Selected GitHub projects
- Venture and project interests
- Blog / field notes
- Contact information

## Local development

This is a dependency-free static site. Serve the repository root with any local HTTP server:

```powershell
python -m http.server 4173
```

Then open `http://localhost:4173`.

## Deployment

The `main` branch is published with GitHub Pages at the user-site domain above.
